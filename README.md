# 📈 실시간 자산 관리 툴 (Currency1)

이 프로젝트는 현재 미쳐 날뛰는 환율실시간으로 분석하기 위해 만들어졌습니다. 


## 🚀 주요 기능
- **실시간 환율 조회**: USD/KRW 환율이 1,500원을 넘으면 경고 메시지를 출력합니다.
